# Student Grading Web Application

A Django-based web application to manage students, courses, lecturers, subjects, and scores.

## Features

- **User Authentication**
  - Register, log in, log out
  - Password change
- **Student Management**
  - List students
  - View student details
  - Assign subjects
  - Track scores
- **Student & Subject Management**
  - CRUD operations for students and subjects
- **Lecturer Management**
  - List lecturers and their emails
- **Score Management**
  - Assign and update scores
  - Average score calculation
- **Dynamic UI**
  - AJAX-based student score details
- **Data Seeding**
  - Faker-based seeding for courses, lecturers, subjects, students, and scores
- **Responsive Design**
  - Built with Bootstrap 5
